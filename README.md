# Image Manager (Backend)

Серверная часть для приема, сохранения и отдачи файлов (картинок). Написана на Node.js с использованием фреймворка Koa.js.
Сервер использует `koa-body` для парсинга `multipart/form-data` и `koa-static` для раздачи сохраненных изображений.

## Запуск проекта локально

1. Установите зависимости:
   ```bash
   npm install

2. Запустите сервер:
   ```bash
   node server.js.
